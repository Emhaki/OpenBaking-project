### 인턴 프로젝트 내역
**오픈뱅킹 API를 활용한 테스트 서비스 개발**
- API 명세분석 및 산출물 작성
    - 분석한 명세를 바탕으로 화면설계, DB설계, 프로그램 명세서 산출물 작성
    - 오픈뱅킹에서 제공하는 10개 API 사용
- 기능구현
    - 사용자 로그인 기능 및 유효성 체크 기능
    - 사용자 토큰발급, 이용기관 토큰발급, 계좌목록조회, 거래내역조회, 잔액조회, 수취조회, 입금이체, 출금이체 테스트 기능 구현
    - 각 기능에 대한 예외처리 구현
