### 인턴 수행 내역
1. **오픈뱅킹 API를 활용한 테스트 서비스 개발**
- API 명세분석
    - id: 결과로 볼 상품코드
    - 2개 이상의 상품을 볼 경우 콤마(,)로 구분. ex: id=1,2,3
- 결과
    - JSON 형태의 결과
    - 입력 상품과 결과 상품을 상품 정보와 조합하여 반환한다.
